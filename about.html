  • This remake of Round 42 was created by Cathal Lenihan as part of his final year project while studying Computer Games Development at the University of Limerick

    It was created using the Unity 3D engine

    The scripting was done using C#

  • It is a remake of the original game made by Elven studios in 1986

    This is the first game made by the studio. The second game was called bad rats and released in 1993.

  • This website was created to show information and updates on the game

  • It was also created to show users information on how they are doing in the game and things like leaderboards
